**Impact: Critical | Category: Security / Agent Frameworks**

### 2. GitHub Ships Squad: Multi-Agent Teams Living Inside Your Repository
**Impact: High | Category: Agent Frameworks / Multi-Agent Systems**

GitHub released Squad, an open-source project built on Copilot that initializes a pre-configured AI agent team directly inside your repo. Two commands (`npm install -g @bradygaster/squad-cli` + `squad init`) spin up a coordinator, frontend dev, backend dev, and tester — all running in parallel with independent 200K-token context windows. Agent memory lives in `.squad/` as plain text files (versioned with git). The coordinator routes natural language tasks to specialists automatically. This is the most concrete implementation of repo-native multi-agent development to date, and it changes the competitive landscape for Claude Code's subagent model.

**Action items for our KB:** Add to best-of-breed-directory.md, update swarm-patterns-by-dev-stage.md with Squad as a new orchestration option.

**Sources:** [GitHub Blog](https://github.blog/ai-and-ml/github-copilot/how-squad-runs-coordinated-ai-agents-inside-your-repository/), [StartupHub](https://www.startuphub.ai/ai-news/technology/2026/github-s-squad-ai-agents-live-in-your-repo)

### 4. MCP Registry Hits 10K+ Servers; Qualys Flags MCP as "Shadow IT for AI"
