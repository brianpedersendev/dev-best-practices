# Research Scope

## Topic: Stock Analyst MCP Server
Building a Python MCP server that gives Claude the ability to analyze stocks for long-term investing, screen opportunities, and maintain a personal investment knowledge base.

## Purpose:
Feed into an implementation plan for a solo developer (beginner investor) who wants a personal AI-powered stock analysis tool integrated with Claude via MCP.

## Key Questions:
1. What free financial data APIs are reliable enough to power stock analysis (fundamentals, screening, company data)?
2. What existing finance/stock MCPs exist, and are any trustworthy enough to use or extend?
3. What's the best architecture for the knowledge base (personal notes + investment frameworks)?
4. What Python libraries are best suited for financial data processing in an MCP context?
5. What investment analysis frameworks are most useful for a beginner long-term investor?
6. How do existing AI-powered finance tools approach the "not financial advice" liability issue?

## Out of Scope:
- Real-time trading or brokerage integration
- Technical analysis / charting
- Options, crypto, forex
- Paid data sources (v1 is free APIs only)
- Multi-user features
- Backtesting or portfolio simulation

## Prior Knowledge:
- Brian is a solo developer, beginner investor with existing stock holdings
- Python is the chosen language (better finance library ecosystem)
- Free APIs only for v1
- Wants: stock lookup, portfolio analysis, screening, bull/bear analysis, knowledge base
- Open to using existing finance MCPs if trustworthy
- This is both a fun project and intended for real use
